# 🤖 Task Manager Agent

> מערכת ניהול משימות חכמה המבוססת על שיחה טבעית עם AI — פשוט תגידי מה את צריכה, והאייג'נט יטפל בשאר.

---

## 🏗️ ארכיטקטורה

```
task_manager_agent/
├── main.py              # FastAPI server + /chat endpoint
├── agent_service.py     # OpenAI agent עם function calling
├── todo_service.py      # לוגיקת ניהול המשימות (CRUD)
├── tools_schema.py      # הגדרת הכלים ל-OpenAI
├── requirements.txt
└── frontend/            # React + Vite (ממשק משתמש)
    └── src/
        └── App.jsx
```

### זרימת הנתונים

```
משתמש → React UI → POST /chat → OpenAI GPT-4o-mini
                                      ↓
                              function calling
                                      ↓
                              todo_service (CRUD)
                                      ↓
                              תשובה בשפה טבעית → משתמש
```

## 🚀 התקנה והרצה

### דרישות מקדימות
- Python 3.9+
- Node.js 18+
- OpenAI API Key

### Backend

```bash
# התקנת תלויות
pip install -r requirements.txt

# הגדרת API Key
set OPENAI_API_KEY=your-api-key-here   # Windows
export OPENAI_API_KEY=your-api-key-here  # Mac/Linux

# הרצת השרת
uvicorn main:app --reload
```

השרת יעלה על: **http://localhost:8000**

### Frontend

```bash
cd frontend
npm install
npm run dev
```

הממשק יעלה על: **http://localhost:3000**

## 💬 דוגמאות שיחה

### ➕ הוספת משימות

```
את:   "הוסיפי משימה: להגיש דוח עד יום ראשון"
🤖:   ✅ המשימה 'להגיש דוח עד יום ראשון' נוספה בהצלחה!

את:   "תוסיפי פגישה עם הצוות ב-10 לחודש, זה דחוף"
🤖:   ✅ נוספה משימה מסוג 'פגישה': פגישה עם הצוות — תאריך: 10 לחודש

את:   "צרי משימה: לקנות מצרכים, תיאור: חלב ביצים לחם"
🤖:   ✅ המשימה 'לקנות מצרכים' נוספה עם התיאור: חלב, ביצים, לחם
```

### 📋 צפייה וסינון משימות

```
את:   "הראי לי את כל המשימות"
🤖:   📋 יש לך 3 משימות:
      1. להגיש דוח עד יום ראשון [ממתינה]
      2. פגישה עם הצוות — 10 לחודש [ממתינה]
      3. לקנות מצרכים [ממתינה]

את:   "מה המשימות שעדיין לא הושלמו?"
🤖:   ⏳ יש לך 3 משימות פתוחות: ...

את:   "הראי לי רק את הפגישות"
🤖:   📅 נמצאה פגישה אחת: פגישה עם הצוות — 10 לחודש

את:   "מה המשימות שלי לשבוע הקרוב?"
🤖:   🗓️ יש לך 2 משימות השבוע: ...
```

### ✏️ עדכון משימות

```
את:   "סמני את המשימה הראשונה כהושלמה"
🤖:   ✅ המשימה 'להגיש דוח' עודכנה לסטטוס: הושלמה!

את:   "שני את התאריך של הפגישה ל-15 לחודש"
🤖:   📝 תאריך הפגישה עם הצוות עודכן ל-15 לחודש

את:   "עדכני את המשימה מספר 3 — הוסיפי תיאור: לא לשכוח קפה"
🤖:   ✅ התיאור עודכן בהצלחה!
```

### 🗑️ מחיקת משימות

```
את:   "מחקי את המשימה מספר 2"
🤖:   🗑️ המשימה 'פגישה עם הצוות' נמחקה בהצלחה

את:   "תמחקי את כל המשימות שהושלמו"
🤖:   🗑️ נמחקו 2 משימות שהושלמו
```

## 🔧 פעולות CRUD נתמכות

| פונקציה | תיאור | פרמטרים עיקריים |
|---------|-------|-----------------|
| `get_tasks` | שליפת משימות עם סינונים | `status`, `task_type`, `start_date`, `end_date` |
| `add_task` | הוספת משימה חדשה | `title`, `description`, `task_type`, `start_date`, `end_date` |
| `update_task` | עדכון שדה כלשהו במשימה | `task_id` + כל שדה לעדכון |
| `delete_task` | מחיקת משימה לפי ID | `task_id` |

### סטטוסים אפשריים

| סטטוס | משמעות |
|-------|--------|
| `pending` | ממתינה לביצוע |
| `in_progress` | בתהליך |
| `done` | הושלמה |

## 📁 קבצים מרכזיים

### `agent_service.py`
מנהל את השיחה עם OpenAI — שולח את הבקשה, מזהה אם נדרשת קריאה לפונקציה, מבצע אותה ומחזיר תשובה בשפה טבעית.

### `todo_service.py`
שכבת הנתונים — שומרת את המשימות בזיכרון ומספקת פעולות CRUD מלאות עם תמיכה בסינון.

### `tools_schema.py`
מגדיר ל-OpenAI אילו פונקציות קיימות ומה הפרמטרים שלהן, כך שה-AI יודע מתי ואיך לקרוא לכל אחת.

### `main.py`
שרת FastAPI עם endpoint יחיד `/chat` שמקבל הודעה ומחזיר תשובה.
